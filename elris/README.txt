ELRIS2D: A MATLAB Package for the 2D Inversion of DC Resistivity Data


ELRIS2D is an open source package written in 
MATLAB using Version 7.10 Release 2010a.

Installation Procedure
---------------------------------------------------------------------------
It is suggested to create a directory called elris in MATLAB working directory

Example: E:\Documents\MATLAB\elris

and unzip all files and folders into this folder.

\img folder contains image files used in the GUI. Do not delete the folder
or any file in it.

\cptfiles folder contains GMT color tables used to display the sections

Type in "elris" in the MATLAB command window for running the program.
